# 关于页面

欢迎来到关于页面，从这里我们会开始打造一个属于你的完美的关于页面。
*开始之前可以前往最下方查看`流殃博客个人页面`查看效果哦~*

## 主要设置

### 展示顺序
用途：设置卡片展示的顺序，一般不修改默认即可。
```
content-idea,authorCareers,personalities-photo,textarea,game,like-music,statistics-map
```
