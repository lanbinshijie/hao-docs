## 安装主题

在 [Release](https://github.com/liuzhihang/halo-theme-hao/releases) 页面下载最新版本 zip 后，通过 Halo Console 后台主题安装处上传即可。

详细文档请参考：[https://docs.halo.run/user-guide/themes](https://docs.halo.run/user-guide/themes)
